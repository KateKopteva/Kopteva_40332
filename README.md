# 🏰 Достопримечательности Беларуси

Веб-приложение для просмотра каталога туристических достопримечательностей Беларуси с поддержкой маршрутов, системой аутентификации, администрированием, REST API и Blazor-клиентом.

## 📖 О проекте

Приложение представляет собой каталог туристических достопримечательностей Беларуси, организованных по маршрутам. Пользователи могут просматривать достопримечательности, фильтровать их по маршрутам, а администраторы — управлять контентом через веб-интерфейс.

### 🎯 Основные возможности

- 📋 **Каталог достопримечательностей** с фильтрацией по маршрутам
- 🗺️ **Туристические маршруты** (Замки Беларуси, Минск, Беловежская пуща и др.)
- 🔐 **Аутентификация и авторизация** на базе ASP.NET Core Identity
- 👨‍💼 **Административная панель** для CRUD-операций
- 🔌 **REST API** для интеграции с внешними клиентами
- 🎨 **Blazor-клиент** для интерактивной работы с API
- 📸 **Загрузка изображений** достопримечательностей
- 📄 **Пагинация** и поиск

## 🛠️ Технологии

### Backend
- **ASP.NET Core 8.0** — основной фреймворк
- **ASP.NET Core MVC** — веб-интерфейс (Kopteva.UI)
- **ASP.NET Core Identity** — аутентификация и авторизация
- **Entity Framework Core 8.0** — ORM для работы с БД
- **Microsoft SQL Server** — реляционная база данных
- **Swagger / OpenAPI** — документация REST API

### Frontend
- **Razor Views** — серверный рендеринг (MVC)
- **Blazor Server** — интерактивный UI (Kopteva.Blazor)
- **Bootstrap 5** — стилизация
- **HTML5 / CSS3 / JavaScript**
